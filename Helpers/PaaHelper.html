Keyboard shortcuts of PA Application

1. Common Shortcuts

  • ESC : To escape Dialog popup
  • Delete : To Delete a record
  • Alt + Y & 'Y' : To confirm action 'Delete' an item
  • Alt + N & 'N' : To cancel action 'Delete' an item
  • Alt + O : To confirm an action on 'Document Complete' dialog
  • Alt + C : To cancel an action on 'Document Complete' dialog

2. Auto Document Shortcuts

  • Alt + C : To open 'Custom Chart'
  • Alt + P : To select a Pie Chart type of 'Custom Chart'
  • Alt + B : To select a Bar Chart type of 'Custom Chart'
  • Alt + L : To select a Line Chart type of 'Custom Chart'
  • Alt + M : To generate a 'Custom Chart'

3. Client Info Shortcuts

  • Alt + A : To create a new Client
  • Alt + C : To cancel creating a new Client
  • Alt + U : To launch a 'Mgmt-Wide Info Update' form
  • Alt + C : To cancel and close 'Mgmt-Wide Info Update' form
  • Alt + S : To open a 'Enter SQL Query' form
  • Alt + O : To confirm search criteria of 'Enter SQL Query' form
  • Alt + C : To cancel and close an 'Enter SQL Query' form
  • Alt + D : To confirm an update record(s) of 'Edit Special Price' form

4. Info Resources Shortcuts

  • Alt + L : To select 'Landlord Contact' tab
  • Alt + C : To select 'Criminal Information' tab
  • Alt + E : To select 'Employment Verification' tab
  • Alt + D : To select 'Document Library' tab

5. Application Explore Shortcuts

  • Ctrl + N : To add a new Application
  • Ctrl + F : To open 'Search App' form
  • Ctrl + A : To open 'Enter Archive' form
  • Ctrl + D : To do spelling check for the current textbox
  • Ctrl + P : To move to a 'Pickup' location
  • Ctrl + R : To move to a 'Review' location
  • Alt + A : To select 'Application Info' tab
  • Alt + C : To select 'Credit Info' tab
  • Alt + E : To select 'Employment Info' tab
  • Alt + R : To select 'Rental Info' tab
  • Alt + I : To select 'Criminal/Eviction' tab
  • Alt + F : To select 'Final Info' tab
  • Alt + M : To select 'Application Menu' tab
  • Alt + T : To select 'Internet Tools' tab
  • Alt + 1 : To show standard comment of 'Standard Verifs. 1' under 'Employment Info' tab
  • Alt + 2 : To show standard comment of 'Standard Verifs. 2' under 'Employment Info' tab
  • Alt + S : To show standard comment of 'Standard Refs.' under 'Rental Info' tab
  • Alt + N : To create a 'New Ref' of 'Rental Info'
  • Alt + D : To delete a 'Ref' of 'Rental Info'
  • Alt + S : To show standard comment of 'Standard Comments' under 'Final Info' tab
  • Alt + P : To open 'Special Criteria Info'
  • Alt + P : To print an Application Report
